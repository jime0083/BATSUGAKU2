プライバシーポリシー

最終更新日: 2026年1月24日

バツガク（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用するかを説明します。

1. 収集する情報

1.1 アカウント情報

  • Googleアカウント情報（メールアドレス、表示名、プロフィール画像）
  • GitHubアカウント情報（ユーザー名、アクセストークン）
  • X（Twitter）アカウント情報（ユーザー名、アクセストークン）

1.2 利用データ

  • GitHubへのpush履歴（日付のみ）
  • 学習ストリーク（連続学習日数）
  • 獲得バッジ情報
  • サブスクリプション状態

1.3 デバイス情報

  • プッシュ通知用トークン（通知を有効にした場合のみ）

2. 情報の利用目的

収集した情報は、以下の目的で使用します：

  • ユーザー認証およびアカウント管理
  • GitHubへのpush状況の確認
  • X（Twitter）への自動投稿（サボりツイート、ストリーク達成ツイート）
  • 学習統計の記録と表示
  • バッジの付与
  • プッシュ通知の送信
  • サブスクリプションの管理

3. 情報の共有

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者と共有しません：

  • ユーザーの同意がある場合
  • 法令に基づく開示要求がある場合
  • サービス提供に必要な外部サービス（Firebase、GitHub API、X API）との連携

4. 外部サービスとの連携

本アプリは以下の外部サービスを利用します：

  • Firebase（Google）: 認証、データベース、クラウド機能
  • GitHub API: push履歴の取得
  • X（Twitter）API: ツイートの投稿

各サービスのプライバシーポリシーについては、それぞれの公式サイトをご確認ください。

5. データの保管

ユーザーデータはFirebaseのセキュアなサーバーに保存されます。アクセストークンは暗号化して保存されます。

6. ユーザーの権利

ユーザーは以下の権利を有します：

  • アカウントの削除を要求する権利
  • 連携サービスの解除（GitHub、X）
  • プッシュ通知の無効化
  • データの開示を要求する権利

7. 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。13歳未満の方は本アプリを使用しないでください。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、アプリ内で通知します。
